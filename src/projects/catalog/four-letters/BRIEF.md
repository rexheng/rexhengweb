# Four Letters — sprite brief

**Project:** Four Letters — word-search puzzle game.
**Card on projects/index.html:** https://four-letters-two.vercel.app/
**Sprite role:** prop. A 4×4 grid of letter tiles — reads instantly as
"word game" from the silhouette.

## Visual target

- 4×4 grid of cube-shaped tiles with single letters on the top face.
- Each tile slightly separated, small bevel at the edges.
- The word "REX " spelled across the first row (one tile each: R, E, X,
  space) so the sprite self-brands without being loud.
- Accent: warm orange (`#d47a42`) — game UI energy without being garish.
- No ability — nothing natural maps to a word-search puzzle.
