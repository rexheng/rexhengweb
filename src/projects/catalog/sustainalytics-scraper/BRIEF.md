# Sustainalytics Scraper — sprite brief

**Project:** Sustainalytics Scraper — Python web scraper for ESG data.
**Card on projects/index.html:** https://github.com/rexheng/sustainlyticsscraper/tree/main
**Sprite role:** prop. A stack of data-pack boxes with a green leaf-mark
on top — reads as "scraped data you can carry".

## Visual target

- Three stacked cardboard-toned boxes of descending size (pallet-of-data feel).
- A small green leaf emblem on the topmost box.
- No ability — a scraping script isn't a physics gag.
- Accent colour `#6b9b5f` (ESG sage green).
